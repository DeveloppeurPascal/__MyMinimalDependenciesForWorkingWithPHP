# Installation des dépendances minimales pour travailler avec PHP sur mes projets

[This page in English.](README.md)

Ce projet sert d'initialisation de mon environnement de travail pour faire des sites web, des backoffices ou des serveurs d'API.

En lançant le fichier batch vous allez rapatrier un certain nombre de dépôts de code dans le dossier dans lequel vous l'exécuterez. Faites le idéalement dans le dossier de votre serveur web local.

Vous pouvez utiliser ce projet de base pour vos propres installations en faisant un fork ou une copie puis en le personnalisant selon vos besoins ou en vous inspirant simplement de son fonctionnement.

Si vous travaillez sur des logiciels, applications mobiles et jeux vidéo en Pascal (sous Delphi ou pas) vous pourriez aussi utiliser mon [autre dépôt de code d'installation minimale pour projets web en Delphi](https://github.com/DeveloppeurPascal/__MyMinimalDependenciesForWorkingWithDelphi).

## Installation des codes sources

Pour télécharger ce dépôt de code il est recommandé de passer par "git" mais vous pouvez aussi télécharger un ZIP directement depuis [son dépôt GitHub](https://github.com/DeveloppeurPascal/__MyMinimalDependenciesForWorkingWithPHP).

Téléchargez ou copiez le fichier "install-sub-repositories.bat" dans le dossier (par exemple dans C:\xampp\htdocs si vous utilisez XAMPP) où vous désirez installer mon arborescence de projets puis exécutez le. "Git" doit être accessible dans le chemin par défaut de votre ordinateur.
